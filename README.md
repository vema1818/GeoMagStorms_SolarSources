The File List_GMS_Storms.csv contains information of geomagnetic storms (Dst < -100 nT) that occurred between 1996-2025. The solar sources are identified by manually verifying the remote and in situ data from different solar observationsThis list will be updated with more events as time progresses. 
The .csv file can be read in IDL with _read_csv.pro_ and analyse the data in each columns of interest
The figures are some of statistical analysis results in the publication in Solar Physics journal
